Build a modern, production-ready web platform named **“Careera”** — an AI-powered event recommendation system designed for students, organizers, and administrators.

The platform should feel like a professional SaaS product (similar to LinkedIn + event platforms), not a generic template. Focus on clean UX, structured layouts, and smooth interactions using Tailwind CSS.

---

### 🎯 CORE CONCEPT

Careera connects students with relevant events (hackathons, workshops, competitions) using intelligent recommendations based on their interests and past participation.

There are **3 roles** in the system:

1. User (Student)
2. Organizer
3. Admin (restricted access)

---

### 🔐 Authentication & Onboarding

* Signup fields:

  * Full Name
  * College Name
  * Interests (multi-select)
  * Past Experience (events/hackathons participated)
* Login system with session handling

---

### 🧠 AI-Based Recommendations

* Show personalized event recommendations based on:

  * Interests
  * Past participation
* Section title: **“Recommended For You”**

---

### 🖌️ Design Guidelines

* Use **Tailwind CSS**
* Clean spacing and grid layout
* Rounded cards (`rounded-2xl`)
* Soft shadows (`shadow-md`)
* Smooth hover effects
* Micro-interactions (scale, fade, slide)

---

### ✨ Animations (IMPORTANT)

* Tab switching → smooth fade/slide
* Card hover → slight scale-up
* Button click → ripple or subtle bounce
* Page transitions → fade-in

---

### 📱 Responsiveness

* Fully responsive:

  * Mobile
  * Tablet
  * Desktop
